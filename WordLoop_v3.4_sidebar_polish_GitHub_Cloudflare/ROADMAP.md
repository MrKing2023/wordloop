# WordLoop 产品路线图

## 已记录，后续实现

### 1. 题型混合

- 每日学习和复习中穿插“英语句子填空”和“中文整句翻译成英文”。
- 不再把两种题型完全分开，避免形成固定答题惯性。
- 可在设置中选择混合比例，例如填空 70% + 整句翻译 30%。

### 2. 填空卡的整句复习

- 一张原本用于填空的卡片，在后续复习阶段可自动转换成中文提示。
- 用户需要手打完整英文句子，而不是只填写目标单词。
- 卡片仍复用原有例句、释义、搭配和复习记录，不重复创建词库数据。

### 3. AI 语义判分

- 对整句翻译答案先进行本地严格匹配。
- 与标准答案不一致时，再提交给服务端 AI 判断：语义是否正确、语法是否成立、表达是否自然。
- AI 返回“可接受 / 基本可接受但需修改 / 不可接受”，并解释差异。
- API 密钥只保存在 Cloudflare Worker 或其他服务端，绝不写入前端代码。
- AI 判分结果需保留可追溯记录，并允许用户手动接受某个新答案。

## 当前版本已实现

- 左侧菜单栏收起与展开，并保存当前浏览器偏好。
- 答题后左侧“上一题”和右侧“下一题”快捷按钮。
- 页面底部继续保留上一题、下一题按钮。
- 返回上一题时恢复该题在当前练习会话中的答案、提示和解析状态，不重复记录学习日志。
