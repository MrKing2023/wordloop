# WordLoop v3.4 更新说明

## 侧栏交互

- 展开状态下，WordLoop Logo、名称和收起按钮分别占位，不再互相遮挡。
- 展开/收起按钮改为双栏面板图标，更接近 ChatGPT 的侧栏操作。
- 收起状态只显示紫色 W Logo。
- 鼠标悬停或键盘聚焦收起状态的 Logo 时，W 会切换为“展开侧栏”图标。
- 点击 Logo 即可重新展开侧栏。
- 收起状态继续保存在当前浏览器。

## 题目导航

- 宽屏左右悬浮按钮改为横向文字：
  - `← 上一题`
  - `下一题 →`
- 修复“上一题”竖排后阅读方向反转的问题。
- 页面底部原有上一题、下一题按钮继续保留。
- 窄屏和移动设备自动隐藏悬浮按钮。
