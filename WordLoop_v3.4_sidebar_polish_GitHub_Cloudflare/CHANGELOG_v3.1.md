# WordLoop v3.1 修复说明

- 修复点击“学习计划”“第 N 天”“继续学习”后地址改变但页面仍停留首页的问题。
- 移除旧 hashchange 路由监听器并绑定 v3 路由。
- 修复“只练错词”覆盖正常学习继续位置的问题，两个模式现在分别保存位置。
- 到期复习不再把尚未学习的新词计算为待复习，也不会在没有到期卡片时随机塞入新词。
- 正常按天学习中，答错或使用提示的卡片会保留到“本日错词”；在专项错词模式答对后才移出错词列表。
- 修复完成一天后再次进入时停在最后一题的问题，已完成学习日会从头开始重学。
- 增加路由错误保护，异常时显示可恢复页面而不是无响应。
- 音标提示优先使用人工词库中的标准音标，避免使用质量较差的自动派生音标。
