## 浏览器集成测试

使用 Chromium 宽屏环境执行了以下流程：

1. 首页正常加载。
2. 左侧栏可收起。
3. 切换页面后收起状态继续保留。
4. 左侧栏可重新展开。
5. 可进入第 1 天并开始学习。
6. 第一题答对后显示右侧“下一题”，第一题不显示“上一题”。
7. 进入并答对第二题后，左侧“上一题”和右侧“下一题”同时显示。
8. 点击上一题后恢复第一题的正确答案及详细解析。
9. 返回上一题不会新增学习日志。
10. 再点击下一题后恢复第二题的正确答案及详细解析。
11. 浏览已答题目不会重复新增学习日志。
12. 页面底部同时保留上一题和下一题按钮。
13. 测试过程中未捕获 JavaScript 页面异常。

## 响应式规则

- 宽屏桌面端：显示左右悬浮导航。
- 宽度小于 1440 px：隐藏悬浮导航，使用底部按钮。
- 宽度小于 760 px：隐藏桌面侧栏，使用底部移动导航。
