# WordLoop 产品路线图

## v4.2 已完成：考研英语首发

- 接入考研英语 4112 条核心基础词条；
- 发布 100 张考研增强学习卡；
- 覆盖熟词僻义、学术论证、逻辑关系、因果表达、阅读方法和数据描述；
- 增加考研基础词库浏览器；
- “核心 200 / CET-6 / 考研英语”分别保存学习计划；
- 对 100 张卡执行唯一 ID、唯一单词、词库归属、填空还原、必填字段和扩展内容检查；
- 保留 CloudBase 标准目录与根目录备用数据路径。

## v4.3：扩充精品学习卡

- CET-6 从 100 张扩展到 500 张；
- 考研英语从 100 张扩展到 500 张；
- 对一词多义词按“词义”拆卡，而不是只按词形拆卡；
- 每个词义生成多个候选句，再依据自然度、词义准确度和填空唯一性筛选；
- 建立 `needs_review.json`，只人工复核异常项；
- 增加“熟词僻义”“逻辑词”“学术动词”等专题学习入口。

## v4.4：混合题型复习

- 每日学习与复习混合句子填空和中文整句翻译；
- 原填空卡在后续复习阶段自动升级为完整汉译英；
- 支持设置题型比例，例如填空 70% + 整句翻译 30%；
- 根据熟练度动态提高输出难度；
- 合理但不同于标准答案的表达可加入个人答案库。

## v4.5：AI 语义判分

- 本地严格匹配失败后才调用服务端 AI；
- 返回“可接受 / 基本可接受但需修改 / 不可接受”；
- 显示语义、语法、搭配和自然度反馈；
- API Key 只保存在 CloudBase 云函数或其他服务端；
- 保存判分记录，支持人工覆盖和审计。

## v4.6：完整数据工程

- 从完整 ECDICT CSV 直接提取全部 `ky` 标签词条，不再受 CET-6 本地子集限制；
- 静态 JSON 分片并迁移到 IndexedDB 缓存；
- 支持按词频、主题、词性、难度和考试重叠情况筛选；
- 增加“仅考研新增词”“与六级重叠词”“熟词僻义”等视图；
- 建立词条来源、版本和内容审核状态追踪。

## v5：IELTS 主题词汇

- 核心词汇、写作表达、听力场景和口语主题；
- 重点训练 collocation、paraphrase 和语境准确性；
- 不使用“雅思官方完整词库”的表述；
- 与 CET-6、考研词库共享基础词典，但保留独立学习卡与学习计划。

## 账号与云同步

- IndexedDB 保存大型词库缓存和离线学习记录；
- CloudBase 数据库保存账号、计划、复习状态和用户答案；
- 支持手机、平板和电脑同步；
- 保留 JSON 导出作为可迁移的本地备份。
