# WordLoop 产品路线图

## v4.1 已完成：CET-6 前两步

### 完整基础词库

- 从 ECDICT 中提取全部 `cet6` 标签词条；
- 共 5407 条，包含单词、音标、中英文释义、考试标签、BNC/现代词频和词形变化；
- 新增基础词库浏览器，支持搜索和分页；
- 明确标注数据范围：不宣称为唯一或官方六级考试词表。

### 首批高质量学习卡

- 发布 100 张 CET-6 增强学习卡；
- 每张卡包含语境填空、准确中文翻译、用法提醒、常见搭配、相关表达、词族和扩展例句；
- 自动检查答案在英文句中恰好出现一次；
- 自动检查挖空前缀、答案和后缀能够完整还原原句；
- 自动检查必填字段、唯一 ID、唯一单词和 CET-6 标签来源。

### 多词库学习

- 支持“核心 200”和“CET-6 精选 100”切换；
- 每个词库分别保存按天学习计划；
- 启用 CET-6 后按默认每天 20 个生成 5 天计划；
- 基础词典层不直接进入练习，避免未经审核的例句污染学习内容。

## v4.2：扩展 CET-6 增强卡

- 把 100 张试验卡扩展到 500 张；
- 为每个词义生成 4–6 个候选英文例句；
- 独立生成中文翻译并进行回译检查；
- 增加填空答案唯一性评分和替代答案分析；
- 对多义词按词义拆卡，而不是仅按单词拆卡；
- 建立 `needs_review.json`，只人工复核异常项。

## v4.3：完整 CET-6 学习内容

- 按高频、重点、低频分层补齐增强卡；
- 增加静态 JSON 分片和 IndexedDB 缓存；
- 词库排序支持词频、主题、词性和难度；
- 增加“只学习六级新增词”“与四级重叠词”等筛选；
- 为已掌握词和薄弱词建立独立学习队列。

## v4.4：混合题型复习

- 每日学习和复习中混合句子填空与中文整句翻译；
- 原填空卡在后续复习阶段自动升级为完整汉译英；
- 支持设置题型比例，例如填空 70% + 整句翻译 30%；
- 根据熟练度动态提高输出难度；
- 对标准答案之外的本地已接受表达建立个人答案库。

## v4.5：AI 语义判分

- 本地严格匹配失败后才调用服务端 AI；
- 返回“可接受 / 基本可接受但需修改 / 不可接受”；
- 显示语义、语法、搭配和自然度反馈；
- 用户可把新的合理表达加入个人可接受答案；
- API Key 只保存在 CloudBase 云函数或其他服务端，绝不进入前端仓库。

## v5：考研英语与 IELTS

### 考研英语

- 完整基础词库；
- 熟词僻义；
- 抽象论述；
- 阅读高频搭配；
- 长难句语境和整句输出。

### IELTS

- 核心词汇；
- 写作表达；
- 听力生活场景；
- 口语主题；
- collocation 与 paraphrase；
- 不使用“官方完整词库”的表述。

## 账号与云同步

- 使用 IndexedDB 保存大型词库缓存和离线学习记录；
- CloudBase 数据库保存账号、计划、复习状态和用户答案；
- 支持手机、平板和电脑同步；
- 保留 JSON 导出作为可迁移的本地备份。
