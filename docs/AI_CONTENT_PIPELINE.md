# AI 学习卡内容生成与质量控制

## 为什么不能一次生成后直接发布

考试词库的难点不是“拿到单词列表”，而是确保每张卡的目标词义、英文自然度、中文翻译和填空唯一性同时可靠。WordLoop 采用候选生成、独立翻译、审稿复核和自动检查的流水线。

## 输入以词义为单位

```json
{
  "word": "address",
  "senseId": "address-v-handle",
  "partOfSpeech": "verb",
  "meaningZh": "处理；应对",
  "definitionEn": "to deal with a problem or situation",
  "examTags": ["cet6", "kaoyan"]
}
```

同一个单词的不同词义分别生成，避免把“地址”和“处理问题”混在同一张卡里。

## 推荐流水线

1. 基于词典事实生成 4–6 个英文候选句；
2. 选择符合目标词义、难度和真实搭配的候选；
3. 对最终英文句单独生成中文翻译；
4. 分析挖空后的其他合理答案；
5. 唯一性低的句子转为整句翻译题，或增加可接受答案；
6. 第二个独立审稿步骤给语法、自然度、词义、翻译和唯一性评分；
7. 自动规则检查字段与结构；
8. 异常内容进入 `needs_review.json`；
9. 每批随机抽查 5%–10%。

## 评分建议

```json
{
  "grammar": 5,
  "naturalness": 5,
  "senseAccuracy": 5,
  "translationAccuracy": 5,
  "blankUniqueness": 4,
  "decision": "approved"
}
```

任何核心评分低于 4 的卡片不进入正式词库。

## 发布分级

- 基础版：词典事实，无生成例句；
- 标准版：候选生成 + 自动校验 + 模型复核；
- 精品版：真实语料辅助 + 多模型复核 + 人工抽样。

建议所有考试词汇达到标准版，高频 500–1000 词升级为精品版。

## AI 语义判分

整句翻译先进行本地严格匹配。只有不同于已知答案时，才请求服务端 AI：

- 判断语义是否保持；
- 检查语法；
- 判断表达自然度；
- 说明与标准答案的差别；
- 返回可接受、需修改或不可接受。

API Key 必须保存在 CloudBase 云函数或其他服务端，不能出现在静态网页中。
